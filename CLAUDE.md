# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 项目概述

这是一个 Python 学习仓库，用于练习算法和复习 Python 基础知识。

## 项目结构

- `复习/` - Python 基础语法复习（变量、函数、类、装饰器等）
- `数据结构和算法/` - 算法练习代码

## 运行代码

```bash
# 激活虚拟环境
.venv\Scripts\activate

# 运行 Python 文件
python 复习/复习.py
python 数据结构和算法/算法体验.py
```

## 代码风格

- 代码文件包含大量注释用于学习笔记
- 使用 `#` 注释掉已完成的练习代码
- 中文注释说明概念要点
